# Linux Admin Mini Project

## Overview
This project demonstrates practical Linux system administration skills using a Zorin OS virtual machine. 
It covers user management, permissions, services, automation, security basics, monitoring, and backups.

## Environment
- OS: Zorin OS (VirtualBox VM)
- User: non-root admin user
- Tools: Bash, systemctl, cron, Git, GitHub

## Features Implemented
- User and group management
- File permissions and ownership handling
- Service monitoring using systemctl
- Backup automation using shell scripting
- Cron job scheduling
- Disk and process monitoring
- Git version control
